Backend project
===============

Express with MongoDB from Mlab.


Deploy
------

Установка зависимостей npm packages

    $ npm install

Запуск сервера для разработки
-----------------------------

    $ npm start

    http://localhost:8082/

Cборка
------

Сборка проекта в продакшен

    $ npm run prod

Тесты
-----

Запуск линтера

    $ npm run lint

Запуск тестов

    $ npm run test
